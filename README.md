# task-manager-backend
first spring project


TEAM TASK MANAGER (FULL STACK PROJECT)

---------------------------------------------------------

## PROJECT DESCRIPTION

This is a full-stack web application built using Spring Boot.
It allows users to create projects, assign tasks, and track progress with role-based access.

------------------------------------------------------------

## FEATURES

* User Signup & Login
* Project Management
* Task Creation & Assignment
* Task Status Update
* Dashboard (Tasks overview)
* REST APIs
* Deployed on Railway

-------------------------------------------------------------

## TECH STACK

Backend: Java, Spring Boot
Frontend: HTML
Database: MySQL
Tools: IntelliJ IDEA, Postman
Deployment: Railway

-------------------------------------------------------------------------

## API ENDPOINTS

User APIs:
POST   /users/signup
POST   /users/login
GET    /users/all

Task APIs:
POST   /tasks/create
GET    /tasks/all
PUT    /tasks/updateStatus

Project APIs:
POST   /projects/create
GET    /projects/all

----------------------------------------------------------

## HOW TO RUN LOCALLY

1. Open project in IntelliJ IDEA
2. Run TaskmanagerApplication.java
3. Open browser:
   http://localhost:8080/
